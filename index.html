CleanUrl 🚀

您的链接净化专家：一键还原混淆网址，浏览更纯粹！

✨ 功能特性

  • ✅ 智能替换全角句号： 自动将链接中的“。”替换为标准的“.”，确保链接格式正确可访问。
  • 🗑️ 高效移除干扰字符： 快速识别并删除链接中插入的“删”等无用字符，还原链接的原始面貌。
  • 📋 支持净化剪贴板内容： 无论是从何处复制的链接，只需粘贴到剪贴板，CleanUrl 即可对其进行净化处理。
  • 🚀 净化并直接访问： 不仅能净化链接并复制回剪贴板，更支持直接在新标签页中打开净化后的链接，省时省力。
  • 🌐 多语言支持： 界面根据浏览器语言自动切换，提供简体中文、繁體中文和英文支持。

📦 安装指南

CleanUrl 支持主流的 Chromium 内核浏览器（如 Chrome, Edge）和 Firefox。安装步骤非常简单：

开发者模式加载 (推荐)

  1. 首先，请前往 GitHub.com/12346free/CleanURL 克隆或下载此仓库到您的本地。
  2. 在浏览器地址栏输入 edge://extensions (Edge) 或 chrome://extensions (Chrome) 或 about:addons (Firefox) 打开扩展程序管理页面。
  3. 在扩展程序页面的右上角（或左侧菜单栏）**开启“开发者模式”**。
  4. 点击“**加载已解压的扩展程序**”（或类似选项）。
  5. 选择您下载的 **CleanUrl/ 文件夹**，即可完成安装。
---

🧪 使用方法

安装完成后，CleanUrl 会成为您浏览器工具栏上的得力助手：

  1. 点击浏览器工具栏上的 **CleanUrl 扩展图标**，打开插件面板。
  2. 将含有“删”或“。”等干扰字符的链接**粘贴到剪贴板**中。
  3. 在插件面板中，您可以选择：
    • 🧹 **【净化剪切板】**：净化链接并将其复制回剪贴板。
    • 🚀 **【净化并访问】**：直接在新标签页中打开净化后的链接。

🔐 权限说明

CleanUrl 仅为实现其核心功能而请求必要的权限：

  • `clipboardRead`, `clipboardWrite`：用于读取和写入剪贴板内容，这是净化链接的核心。
  • `tabs`：用于打开新的标签页访问净化后的链接。
  • `scripting`：用于未来功能扩展（目前保留）。

我们承诺您的隐私安全，CleanUrl 不会收集任何个人数据。
